# Vector Editor FRONTEND (vibe-editor)

Простой векторный редактор на Vue 3 + TypeScript + Canvas 2D.

Минимально жизнеспособная версия (MVP) с поддержкой создания, перемещения и редактирования базовых фигур: прямоугольник, круг/эллипс, линия.

## Основные возможности

- Инструменты: выделение, прямоугольник, круг, линия
- Перемещение фигур перетаскиванием
- Выделение и редактирование свойств (позиция, размер, цвета, толщина линии)
- Динамическая отрисовка на `<canvas>`
- Адаптивный размер холста
- Реактивная панель свойств

## Технологический стек

- Vue 3 (Composition API)
- TypeScript
- Pinia (state management)
- Canvas 2D API
- CSS Grid (layout)

## Структура проекта

```
frontend/
├── src/
```

## Установка и запуск

Проект находится в поддиректории `frontend/`. Все команды нужно выполнять из этой папки.

```bash
# 1. Перейти в директорию проекта
cd frontend

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev

# Откроется по адресу http://localhost:5173 (или другой порт, указанный в терминале)
```

### Другие полезные команды

```bash
# Сборка production-версии
npm run build

# Предпросмотр собранной версии локально
npm run preview

# Запуск линтера (если настроен eslint)
npm run lint
```

## Текущий статус

- Работают базовые фигуры: rect, circle, line
- Поддерживается drag & drop, выделение, базовое редактирование свойств
- Нет: поворота, масштабирования, undo/redo, групп, слоёв, экспорта, импорта, snapping, зума

## Планы развития (roadmap)

- Поддержка поворота и масштабирования
- Многоугольники и кривые (path)
- Undo/redo
- Экспорт в SVG / PNG
- Зум и панорамирование холста
- Слои и порядок отрисовки

## Запуск в production

После сборки (`npm run build`) содержимое папки `dist/` можно развернуть на любом статическом хостинге (Netlify, Vercel, GitHub Pages, Nginx и т.д.).
